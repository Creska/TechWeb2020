Come funziona l'editing del CSS

  • L'organizzazione dello stylesheet si basa ovviamente sul sistema di tag HTML e di selettori del CSS. Se ne consiglia quindi l'utilizzo esclusivamente agli utenti che abbiano una minima conoscenza di markup per documenti web.
  • Il file CSS modificabile è totalmente facoltativo (e quindi vuoto), dato che il player è già provvisto di un apposito stylesheet di default. Spetta quindi all'autore aggiungere tutte le regole desiderate.
  • Non vi è bisogno dell'utilizzo degli id. Per selezionare particolari elementi basterà concentrarsi sulle classi e sul sistema di parentela dei selettori CSS.
  • Oltre, ovviamente, al gioco in generale, è possibile stilizzare una singola quest, attività, sottosezione o elemento. In particolare, le classi built-in seguono questo schema:
                            [Body]
                            StoryTitle /* titolo della storia */
                            
                            Quest0 /* container di tutti gli elementi figli della quest. Il numero si riferisce all'indice che questa quest ha nell'apposito array */
                                Activity0 /* container di tutti gli elementi figli dell'attività. Il numero si riferisce all'indice che questa attività ha nell'apposito array */
                                    ActivityText /* container dei paragrafi di testo, gallery ed immagini singole */
                                        TextParagraph
                                        ImageGallery
                                        SingleImage
                                    AnswerField /* container del campo di risposta */
                                        AnswerFieldDescription /* eventuale breve indicazione/consiglio per il giocatore */
                                        AnswerFieldInput /* campo risposta vero e proprio */
                                    NextActivity /* pulsante da cliccare per andare all'attività successiva */
                                    CloseGameBtn /* pulsante per chiudere il gioco - mostrato solo nell'ultima attività */
